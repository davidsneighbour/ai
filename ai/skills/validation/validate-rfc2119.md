# Validate RFC 2119 keywords

Check whether uppercase RFC 2119 words are used consistently.

* `MUST` and `MUST NOT` indicate mandatory requirements.
* `SHOULD` indicates strong recommendation.
* `MAY` indicates optional behaviour.

Flag misuse where wording does not match the intended force.
